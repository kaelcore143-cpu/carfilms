# AI PROJECT STANDARD STRUCTURE

Este archivo está diseñado para que cualquier IA (ChatGPT, Claude, Copilot, etc.) genere proyectos **ordenados, profesionales y consistentes** sin romper el diseño ni la estructura.

La IA debe seguir TODAS estas reglas sin excepción.

---

# 1. OBJETIVO DEL PROYECTO

El objetivo es crear proyectos web modernos con:

* Diseño premium
* Código limpio
* Estructura profesional
* Responsive perfecto
* Animaciones suaves
* Consistencia visual en todos los proyectos
* Efecto parallax
* Iconos svg en vez de emojis

---

# 2. ESTRUCTURA OBLIGATORIA DEL PROYECTO

La IA SIEMPRE debe generar los archivos con esta estructura exacta:

```
project-name/
│
├── index.html
├── style.css
├── script.js
│
├── assets/
│   ├── img/
│   ├── icons/
│   ├── videos/
│   └── fonts/
│
├── components/
│   ├── navbar.html
│   ├── footer.html
│   └── buttons.css
│
├── preview/
│   └── preview.png
│
└── README.md
```

La IA NO debe cambiar nombres de carpetas.
La IA NO debe crear archivos innecesarios.
La IA debe mantener el orden limpio.

---

# 3. REGLAS DE DISEÑO (OBLIGATORIAS)

Todos los proyectos deben seguir este estilo visual:

### Estilo general

* Diseño moderno
* Diseño elegante
* Diseño premium
* Estilo juvenil
* No diseño básico
* No estilo antiguo

### Colores

* Fondo oscuro o fondo limpio
* Contraste fuerte
* Botones llamativos
* Evitar colores apagados

### Tipografía

* Fuente moderna
* Texto limpio
* Buena jerarquía visual

---

# 4. SCROLLBAR OBLIGATORIO

La IA SIEMPRE debe incluir este scrollbar en el archivo style.css

```
/* ===== Premium Scrollbar ===== */

::-webkit-scrollbar {
    width: 8px;
}

::-webkit-scrollbar-track {
    background: #0f0f0f;
}

::-webkit-scrollbar-thumb {
    background: linear-gradient(180deg, #ff2a2a, #ff0000);
    border-radius: 10px;
}

::-webkit-scrollbar-thumb:hover {
    background: #ff0000;
}
```

No se debe cambiar el estilo del scrollbar.

---

# 5. ESTRUCTURA VISUAL OBLIGATORIA DE LA WEB

La IA debe generar la página en este orden exacto:

```
1. Navbar elegante
2. Hero section impactante
3. Servicios / características
4. Sección visual (imágenes grandes)
5. Testimonios
6. Llamado a la acción
7. Footer limpio
```

No cambiar el orden.
No eliminar secciones.

---

# 6. REGLAS DE CÓDIGO

La IA debe cumplir estas reglas:

* Código limpio
* Código bien organizado
* No repetir código innecesario
* HTML ordenado
* CSS separado
* JavaScript separado

La IA NO debe escribir código desordenado.
La IA NO debe escribir estilos dentro del HTML si no es necesario.

---

# 7. REGLAS RESPONSIVE (MUY IMPORTANTE)

El proyecto debe verse perfecto en:

* Móvil
* Tablet
* Laptop
* Pantallas grandes

La IA debe optimizar especialmente para móvil.

---

La IA debe crear un README.md con esta estructura:

```
# Nombre del proyecto

## Descripción

## Características

- Código limpio

## Tecnologías usadas
- HTML

- JavaScript

## Vista previa
Imagen del proyecto

## Autor
Kael Core
Frontend Developer
```

No cambiar la estructura del README.

---

# 9. OBJETIVO FINAL

El objetivo no es solo crear una página web.

El objetivo es crear proyectos que:

* Se vean profesionales
* Sirvan para clientes reales
* Sean rápidos de modificar
* Sean reutilizables
* Sean visualmente superiores a la mayoría de páginas básicas

---

# 10. INSTRUCCIÓN FINAL PARA LA IA

Si alguna parte del diseño no parece profesional, la IA debe mejorarla automáticamente sin preguntar.

Si el diseño se ve básico, la IA debe hacerlo más premium.

La prioridad SIEMPRE es:

**Diseño premium + Código limpio + Estructura profesional**
