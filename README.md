# Carfilms Estética Vehicular

## Descripción
Sitio web premium para Carfilms Estética Vehicular, empresa especializada en polarizado de autos, cerámico de pintura, detailing y vinilado en Medellín, Colombia. Diseño moderno con experiencia de usuario excepcional y optimizado para todos los dispositivos.

## Características
- Diseño premium
- Responsive
- Animaciones suaves
- Código limpio
- Efecto parallax
- Iconos SVG
- Scrollbar personalizado
- Optimización SEO

## Tecnologías usadas
- HTML
- CSS
- JavaScript
